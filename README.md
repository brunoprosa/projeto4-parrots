# Jogo da Memória

Jogo da memória desenvolvido para navegador, funcionando totalmente offline. O jogador deve encontrar todos os pares de cartas no menor número de tentativas possível.

## Funcionalidades

- Interface simples e intuitiva;
- Embaralhamento aleatório das cartas;
- Verificação automática de pares;
- Contagem de jogadas;
- Cronômetro para medição do tempo da partida;
- Funcionamento offline, sem dependência de servidores ou APIs.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Objetivo

Este projeto foi desenvolvido para praticar lógica de programação, e criação de interações dinâmicas com JavaScript.
